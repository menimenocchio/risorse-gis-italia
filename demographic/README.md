## Standard Dataset
Ogni dataset deve includere:
- Data di riferimento
- Livello territoriale
- Variabili disponibili 
- Formato dati
- Licenza d'uso
- Collegamenti a metadati

## Formati
- CSV (dati tabulari)
- Shapefile (dati geografici)
- JSON/GeoJSON (web services)
